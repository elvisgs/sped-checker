# SPED Checker

Verifica a estrutura de arquivos SPED Fiscal/Contribuições:
- Bloco ou registro não suportado
- Linha com número de campos diferente do esperado

## Requisitos

- Node >= 0.12

## Instalação

```shell
$ npm install -g sped-checker
```

## Uso

```shell
$ sped-checker <caminho_arquivo_sped>
```
